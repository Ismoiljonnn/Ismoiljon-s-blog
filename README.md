# ismoiljon.uz | Personal Digital Home 🌿

Mening shaxsiy platformam. Bu yerda men tajribalarimni baham ko'raman, startaplarimni namoyish etaman va minimalist dunyoqarashimni matnga ko'chiraman.

## 🏗 Platforma tarkibi

### ✍️ Blog
"Minimalizm va Paradokslar" falsafasi asosidagi yozuvlar. 
- Ortiqcha "suv" va keraksiz sifatlarsiz quruq, lekin o'tkir matnlar.
- Har bir post o'quvchini o'ylantirishga va xulosa chiqarishga yo'naltirilgan.
- **Uslub:** To'g'ridan-to'g'ri "zarba" bilan boshlanadigan hikoyalar.

### 💼 Portfolio
Men amalga oshirgan va hozirda ustida ishlayotgan loyihalar markazi.

## 🛠 Texnik qism
Sayt maksimal darajada yengil va tez ishlashi uchun qurilgan:
- **Core:** Vanilla HTML5, CSS3, JavaScript.
- **Design:** Doimiy Light Mode, vizual bo'shliq (white space) va skanerlashga qulay iyerarxiya.
- **Hosting:** GitHub Pages / Render.
